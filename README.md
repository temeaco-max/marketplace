# Marketplace

This is a Node.js + React marketplace application.

## Project Structure
- `/client`: React frontend.
- `/server`: Node.js backend.
- `/client/src`: Source files for React.
- `/client/public`: Static files for React.
- `/server/src`: Source files for Node.js.

## Getting Started
1. Clone the repository.
2. Navigate to the `client` directory and run `npm install` to install the frontend dependencies.
3. Navigate to the `server` directory and run `npm install` to install the backend dependencies.
4. Start the server and client separately.
